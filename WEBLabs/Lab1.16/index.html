Робота з обробниками подій в JavaScript

Обробка кліків на зображення

Реалізуйте функціонал, при якому клік на зображення виводить його шлях (атрибут src) у вікні alert.

Динамічний title для посилань

При наведенні на посилання записуйте його текстовий вміст у атрибут title.

Посилання номер один Посилання номер два Посилання номер три

Відображення href посилань

При наведенні на посилання додавайте його href у круглих дужках після основного тексту.

Перший розділ Другий розділ Третій розділ

Одноразова подія для посилань

Додавання href має відбуватися лише при першому наведенні, після чого обробник події видаляється.

Сторінка 1 Сторінка 2 Сторінка 3

Відстеження вмісту полів вводу

При втраті фокусу полем вводу виводити його значення у спеціальний блок.

Одноразовий alert для полів вводу

При першому кліці на поле вводу показувати його значення через alert (лише один раз).

Обчислення квадрату числа

При кліці на число у параграфі замінювати його на квадрат цього числа.

4

6

8

Валідація довжини вводу

При втраті фокусу перевіряти, чи відповідає довжина введеного тексту очікуваній (вказано в data-length).
