Лабораторна робота 1.15

Робота з циклами та умовними операторами в JavaScript

Визначення простих чисел

Використовуючи цикл while, виведіть усі прості числа в діапазоні від 0 до 100.

Послідовність чисел

За допомогою циклу do...while виведіть числа від 0 до 10 з позначенням їх парності.

Операції ділення

Маємо число 10000. Виконуйте його ділення на 2 доти, доки результат не стане меншим за 50.

Визначення пори року

Введіть номер місяця (1-12) для визначення його назви та відповідної пори року.

Температурний конвертер

Введіть температуру у градусах Цельсія для переведення її у градуси Фаренгейта.

Визначення дня тижня

Введіть номер дня тижня (1-7) для відображення його назви.
